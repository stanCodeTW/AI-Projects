This webpage shows the strange behaviour of key events in different browsers. For most cheap keyboard, no more than 3 keys can be pressed down simultaneously.
Some keys remain red when released, as no corresponding keyup event is received. Some keys are never highlighted because the keycode is inconsistent across browsers. Alpha numeric keys are the only safe default set, but by letting the user to set manually, any key can be used.
It is impossible to override some browser's special function key and operating system's key combination, like alt+tab or ctrl+tab, but some including F1~F12 and ctrl+f can be overrided.
